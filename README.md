# Gift Now

Frontend приложение за Giftnow.bg – сайт за подаръци.

## Какво е направено

- **Начална страница** – промо, hero, корпоративни подаръци
- **Вход** – email + парола (Cognito или mock)
- **Продуктов каталог** – мрежа с продукти след логин
- **Админ панел** – регистрация на потребители (само за admin)

## Технологии

- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS**
- **Radix UI / shadcn/ui**
- **Redux Toolkit**
- **AWS Amplify** (Cognito) или mock auth

## Стартиране

```bash
cd client
npm install
npm run dev
```

Отвори [http://localhost:3000](http://localhost:3000).
