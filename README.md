# Gift Now

## Структура на проекта

```
gift-now/
├── backend/
└── client/              # Next.js приложение
    ├── public/
    ├── src/
    │   ├── app/
    │   ├── components/
    │   ├── hooks/
    │   ├── lib/
    │   ├── state/
    │   └── types/
    ├── server/
    └── ... (config файлове)
```

## Стартиране

```bash
cd client
npm install
npm run dev
```

Отвори [http://localhost:3000](http://localhost:3000) в браузъра.

Подробна документация – виж `client/README.md`.
